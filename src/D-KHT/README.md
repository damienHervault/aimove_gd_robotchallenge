### DESCRIPTION
	- A fast Hough Transform implementation, proposed by Vera et al. 

### REQUIREMENTS
	- Qt Creator 5.5.0 or later;
	- GCC 4.9.1 / MSVC 2013 or later; 

### USAGE: 
	- The proper parameters must be set in settings.h header file.
